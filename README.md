# LTS Beta v0.5.6.11 — reprise automatique de la file

- Toute nouvelle entrée de file programme une reprise automatique.
- Une file existante est reprise à l’ouverture de la PWA.
- Une file existante est reprise quand l’application redevient visible.
- Les limites de tentatives et la temporisation restent actives.

Aucun changement Apps Script n’est nécessaire.
